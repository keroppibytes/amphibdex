# Amphibdex 🐸

> A Pokédex for amphibians. Browse, search, and rank 
> frogs, salamanders, and caecilians.

Built in public. 2 hours a day. Every commit documented.

## Stack
- FastAPI
- SQLite  
- Vanilla JS
- Deployed on Vercel

## Running locally
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
